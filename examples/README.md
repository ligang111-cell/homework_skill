# 科技论文写作模板

这是一个用于撰写中文综述论文的 LaTeX 工作目录。复制本文件夹并改名后，先确认写作要求，再开始写作。

## 怎么用

1. 修改 `docs/main.tex` 中的题目、作者信息、关键词等内容。  
2. 先检索并下载真实论文到 `refs/papers/`，再把允许引用的论文登记到 `refs/notes/paper_index.md`。未下载的论文不要引用。  
3. 在 `docs/sections/` 中按顺序写作，标题层级统一使用 `1.`、`1.1`、`1.1.1`。  
4. 图片先留占位，后续把参考论文中的图片整理到 `assets/`。  
5. 数学公式必须来自教材或论文，并保留可追溯来源。  
6. “总结与展望”或“结论”部分统一采用总—分—总结构，中间分点使用“（1）（2）（3）”书写。  
7. 完成后检查：摘要与正文一致、全文主题统一、每篇文献最多引用 2 次、每张图必须且只能被引用 1 次。

## 目录说明

- `docs/`：论文源码，入口文件是 `main.tex`
- `docs/sections/`：摘要、引言、正文、结论等分节文件
- `refs/papers/`：已下载论文 PDF
- `refs/notes/paper_index.md`：允许引用的论文清单
- `refs/bib/references.bib`：BibTeX 文献库
- `assets/`：论文中使用的图片

## 编译

在 `docs/` 目录下执行：

```bash
xelatex main.tex
biber main
xelatex main.tex
xelatex main.tex
```

## 写作建议

- 先定题、再建框架、后写正文，不要一上来就堆字。
- 综述应围绕一个清晰主题展开，删除与主题关系弱的内容。
- 不要出现“从课程内容看”“课程要求表明”等表述。
- 最后按审稿人的标准通读一遍：看逻辑、看层次、看语言是否清楚顺滑。
