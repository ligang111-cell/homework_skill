# 文献综述写作要求：总结

> 来源：`05- 学位论文写作.ppt`。本文仅提取与“大作业：文献综述写作”直接相关的内容，便于后续写作、排版和检查。

---

## 一、大作业基本要求

本次大作业要求按照课堂讲授的“文献综述写作”要求，提交一篇自己选定题目的文献综述。写作时应注意以下几点：

1. 文章应采用论文格式撰写，并包含文献综述的四个基本部分。
2. 正文字数控制在 4000—8000 字之间，过多或过少都会影响成绩。
3. 文章需要图文并茂，不能只有文字堆砌。
4. 参考文献至少引用 30 篇，一般不超过 50 篇。
5. 近 5 年以内的文献数量应不少于参考文献总数的一半。
6. 参考文献格式必须规范，可参照《机械工程学报》投稿指南或 GB/T 7714—2015 文后参考文献著录规范。
7. 论文排版格式应参考《机械工程学报》论文模板。
8. 作业命名规则为：`学号-姓名-科技写作大作业`。

---

## 二、文献综述的概念与作用

文献综述是作者围绕某一专题，对有价值的文献内容、结论、方法和观点进行综合加工、归纳整理和分析评价后形成的文章。它不是简单罗列文献，而是在广泛阅读和理解已有研究的基础上，对某一研究领域的研究现状、发展动态、存在问题和未来趋势进行综合说明。

文献综述也可以称为文献阅读报告，英文常对应 survey、overview、review。它的核心是“综”和“述”：

- “综”是对大量文献进行归类、提炼、概括和整理；
- “述”是在整理基础上进行分析、比较、评价，并提出自己的认识。

文献综述的主要作用包括：

1. 总结和综合前人在该方向已经完成的研究工作。
2. 帮助了解当前研究水平和技术发展状态。
3. 分析已有研究中存在的问题、不足和争议。
4. 指出可能的研究问题、研究空白和发展方向。
5. 为开题报告、论文写作和后续研究提供依据。
6. 汇总重要信息，指导科研选题和研究决策。

---

## 三、文献综述的类型

文献综述一般可分为两类：

### 1. 大综述

大综述通常关注整个领域、专业方向或较大的研究主题。它的特点是范围广、立意高、覆盖面宽，适合初学者了解某一领域的整体发展状况。但由于范围较大，通常难以做到非常深入。

### 2. 小综述

小综述通常围绕较小的研究方向、具体问题、具体算法或具体技术展开，内容更具体、更深入。毕业论文、开题报告或课程论文中的文献综述通常属于“小综述”。

本次大作业更适合写“小综述”，即围绕一个明确、具体、可分析的问题展开，重点说明：

1. 现有研究状况如何；
2. 已有研究还缺在哪里；
3. 未来可能的发展方向是什么；
4. 自己的选题或分析角度有什么意义。

## 四、选题原则

选择文献综述题目时，应遵循以下原则：

### 1. 价值性原则

题目应具有理论价值或实践价值。也就是说，综述的主题不能过于随意，应能体现某一领域的研究意义、工程意义、应用价值或学术价值。

### 2. 新颖性原则

选题应尽量体现新的内容、新的方法或新的分析角度。文献综述不一定要提出原创理论，但应能反映该方向的最新研究进展和发展趋势。

### 3. 具体性原则

题目不能过于空泛，也不能过于狭窄。题目过大时，文献数量庞大，难以深入分析；题目过小时，可能缺乏足够文献支撑，内容难以展开。

### 4. 量力性原则

选题应结合自身知识基础、资料获取条件、写作时间和分析能力，保证能够在规定时间内完成。

### 5. 推荐做法

初次撰写文献综述时，题目宜小不宜大。建议围绕某一具体技术、方法、应用场景或研究问题展开。例如，比起写“机器人研究综述”，更适合写“足式机器人强化学习运动控制方法研究综述”。

## 五、文献综述的基本流程

文献综述写作通常包括以下步骤：

### 1. 选题

确定一个具体、有价值、可完成的研究主题。题目应紧扣文献综述内容，避免题目过大、内容发散或文不对题。

### 2. 搜集和阅读文献

围绕选题系统查找文献资料。文献来源不应只局限于期刊论文，也可以包括：

1. 学术期刊论文；
2. 学术专著、编著及相关著作；
3. 专利文献；
4. 科技报告；
5. 学术会议论文集；
6. 学位论文；
7. 标准、规范和技术文档。

### 3. 文献筛选与整理

对文献进行归纳、分类、比较和分析。重点筛选具有代表性、科学性、可靠性和创新性的文献。文献质量会直接影响综述质量。

整理文献时，应重点关注：

1. 文献的研究背景；
2. 研究目标；
3. 使用的方法、模型、工具或技术路线；
4. 实验条件和数据来源；
5. 主要结果和结论；
6. 创新点；
7. 局限性；
8. 与自己综述主题的关系。

### 4. 拟定提纲

在正式写作前应先列提纲。提纲可以保证文章结构清晰、逻辑连贯、字数分配均衡。不要一开始就直接写正文。

### 5. 成文

根据提纲完成全文写作。写作时不能只堆砌文献，应做到分类清楚、观点明确、评价合理、结论清晰。

## 六、文献综述的四个基本组成部分

本次大作业要求“包含四个部分”，可按文献综述的标准结构组织：

### 1. 绪言

绪言主要说明写作目的、研究背景、核心概念、综述范围、当前研究现状或争论焦点。绪言应让读者对全文将要讨论的问题有初步了解。

绪言可包括以下内容：

1. 研究背景；
2. 研究意义；
3. 相关概念和定义；
4. 综述对象和范围；
5. 国内外研究现状概述；
6. 主要问题或争议；
7. 本文的综述思路和结构安排。

### 2. 主体部分

主体部分是文献综述的核心。其写法可以灵活安排，但必须能够清楚表达综述内容。常见写法包括：

1. 按时间顺序综述：说明某一研究方向的发展历程；
2. 按研究问题综述：围绕不同问题分别展开；
3. 按技术路线综述：比较不同方法、模型或算法；
4. 按研究观点综述：比较不同学者或不同流派的观点；
5. 按应用场景综述：分析技术在不同场景中的应用。

主体部分应做到：

1. 对文献进行归纳、整理和分析比较；
2. 说明研究主题的历史背景、现状和发展方向；
3. 对代表性强、科学性强、创造性强的文献进行重点引用和评述；
4. 避免文献简单罗列；
5. 在“述”的基础上体现“评”。

### 3. 总结或结论

总结部分应根据主体部分的分析结果提出结论。该部分应准确、完整、明确、精练，最好能提出自己的见解。

总结部分一般包括：

1. 相关研究已经取得的主要成果；
2. 目前仍未解决的问题；
3. 解决这些问题的可能关键点；
4. 未来发展趋势和研究方向；
5. 对本综述主题的总体评价。

### 4. 参考文献

参考文献是文献综述的重要组成部分，用于体现研究基础、说明资料来源、尊重他人学术成果，并便于读者检索。

本次大作业参考文献要求：

1. 至少 30 篇；
2. 一般不超过 50 篇；
3. 近 5 年文献不少于一半；
4. 文献格式规范；
5. 正文引用处必须标注参考文献编号；
6. 文后参考文献应按正文引用顺序连续编码。

## 七、写作重点与难点

文献综述的难点不在于“综”，而在于“述”和“评”。

“综”主要是根据大量文献进行分类、提炼和概括；“述”和“评”则要求作者在理解文献的基础上，分析不同研究之间的联系、差异、优缺点和发展趋势。

高质量文献综述应体现：

1. 对文献的充分阅读和理解；
2. 对研究脉络的清晰梳理；
3. 对代表性成果的准确概括；
4. 对不同方法或观点的比较分析；
5. 对问题、不足和未来方向的判断；
6. 作者自己的思考和见解。

低质量文献综述常见问题包括：

1. 只罗列文献，没有归纳；
2. 只有介绍，没有评价；
3. 引用文献过旧；
4. 文献来源不权威；
5. 结构混乱，逻辑不清；
6. 综述范围过大或过小；
7. 没有提出研究空白或发展趋势；
8. 正文引用和参考文献格式不规范。

## 八、文献选择与引用注意事项

### 1. 优先选择的文献

写作时应优先选择：

1. 权威文献；
2. 经典文献；
3. 近几年发表的文献；
4. 高被引文献；
5. 与主题高度相关的文献；
6. 自己真正阅读和使用过的文献。

### 2. 少用或不用的文献

应少用：

1. 教材；
2. 百度知道、论坛、博客等未经严格筛选的信息；
3. 自己没有读过、只是从别的文章参考文献中看到的“第二手文献”；
4. 与主题关系不大的文献。

### 3. 近五年文献的重要性

文献综述的主要对象应是近年的研究，特别是近五年以内的文献。因为文献综述的重要任务之一，是弄清楚当前最有发展价值的研究工作和研究方向。

### 4. 写作前应再次补充近期文献

即使已经完成大部分写作，在论文定稿前也应再次检索近期文献，防止遗漏最新成果。

## 九、图、表、公式的使用要求

本次大作业要求“图文并茂”，因此建议合理使用图、表和必要公式。

### 1. 图的要求

图应具有“自明性”，即读者只看图题和图中标注，也能大致理解图表达的含义。

图的基本要求：

1. 统一编号；
2. 有图题；
3. 图题置于图下方；
4. 图中文字、符号和单位清楚；
5. 图必须与正文内容相关；
6. 正文中应引用和解释图，不能只放图不说明。

### 2. 表的要求

表也应具有“自明性”。

表的基本要求：

1. 统一编号；
2. 有表题；
3. 表题置于表上方；
4. 通常使用三线表；
5. 表格内容应简洁、清楚；
6. 正文中应对表格进行说明和分析。

### 3. 公式的要求

如果文章中使用公式，应注意：

1. 公式应分章节依次编号；
2. 公式中的变量和符号应解释清楚；
3. 度量单位应使用国际单位制；
4. 如果引用非国际单位制，应换算后加注；
5. 公式必须服务于综述主题，不能为了凑形式随意加入。

## 十、摘要、关键词、引言和结论写法

虽然本次任务是文献综述，但排版可参考论文格式，因此摘要、关键词、引言和结论也应规范。

### 1. 题名

题名应准确、得体、简短、规范、醒目。题目必须紧扣论文内容，做到题文一致。题目一般不宜超过 20 个字，应避免使用生僻缩略语、字符代号或公式。

### 2. 摘要

摘要不是写作意义，也不是引言。摘要应简明说明论文的基本内容和方法。

摘要一般包括四个要素：

1. 目的：说明研究范围和目的；
2. 方法：简述综述采用的文献分析思路、分类方法或研究路径；
3. 结果：概括已有研究的主要进展、成果和问题；
4. 结论：给出综述得出的主要认识和未来方向。

摘要应避免：

1. 照搬目录；
2. 照搬结论；
3. 内容过长；
4. 自我评价；
5. 列举大量例证；
6. 使用图表。

### 3. 关键词

关键词一般列出 3—5 个，应与论文内容关系紧密，并与论文题目有直接联系。关键词可不完全等同于题目中出现的词。

### 4. 引言

引言又称前言，是全文的引论部分。可包括：

1. 研究背景；
2. 研究目的和意义；
3. 前人研究情况及存在问题；
4. 本文要解决或讨论的问题；
5. 本文采用的综述方法和结构安排。

引言应避免与摘要重复，也不能成为摘要的扩写。

### 5. 结论

结论是全文的概括和总结，应体现文章的精华。

结论可包括：

1. 已有研究说明了什么问题；
2. 得出了哪些规律或共识；
3. 解决了哪些理论或实际问题；
4. 对前人研究有哪些补充、修正或发展；
5. 当前研究还存在哪些不足；
6. 后续研究的关键点和方向是什么。

## 十一、参考文献格式要求

参考文献应按正文中引用出现的先后顺序连续编码，并将序号置于方括号中。正文中引用多篇文献时，应将各篇文献序号全部列出，序号之间用逗号分隔；连续编号可用短横线或波浪线连接。例如：

- 张某某[1]对该问题进行了研究。
- 相关方法见文献[2]。
- 早期研究结果[3,4]表明……
- 相关理论可参考文献[5-9]。

### 1. 专著、论文集、学位论文、报告

格式：

```text
[序号] 主要责任者. 文献题名[文献类型标识]. 版本. 译者. 出版地: 出版者, 出版年: 起止页码.
```

常用文献类型标识：

- 专著：[M]
- 期刊论文：[J]
- 论文集：[C]
- 学位论文：[D]
- 报告：[R]

示例：

```text
[1] 喻萍等. 实用信息资源检索与利用[M]. 北京: 化学工业出版社, 2005: 45-52.
[2] 朱建立. 面向对象的分布式知识处理系统[D]. 北京: 中国科学院计算技术研究所, 1987: 15-20.
```

### 2. 期刊论文

格式：

```text
[序号] 主要责任者. 文献题名[J]. 刊名, 年, 卷(期): 起止页码.
```

示例：

```text
[3] 何绍华. 软件产业标准化与质量管理初探[J]. 现代图书情报技术, 2002(5): 93-95.
```

### 3. 论文集中的析出文献

格式：

```text
[序号] 析出文献主要责任者. 析出文献题名[A]. 论文集主要责任者. 论文集名[C]. 出版地: 出版者, 出版年: 起止页码.
```

示例：

```text
[4] 彭韶兵. 国家财务经营公司构建的思考[A]. 2002中国会计教授年会论文集[C]. 成都: 西南财经大学出版社, 2002: 252-256.
```

### 4. 国际、国家标准

格式：

```text
[序号] 标准起草者. 标准编号, 标准名称[S]. 出版地: 出版者, 出版年.
```

示例：

```text
[6] 全国量和单位标准化技术委员会. GB3100-3102—93, 量和单位[S]. 北京: 中国标准出版社, 1997.
```

### 5. 专利

格式：

```text
[序号] 专利所有者. 专利题名[P]. 专利国别: 专利号, 出版日期.
```

示例：

```text
[7] 司信. 用无菌蝇蛆制备甲壳素的方法[P]. 中国专利: 98103446, 1998-07-30.
```

### 6. 电子文献

格式：

```text
[序号] 主要责任者. 电子文献题名[电子文献类型标识/载体类型标识]. 电子文献出处或可获得地址, 检索日期.
```

常见电子文献类型标识：

- 数据库：[DB]
- 计算机程序：[CP]
- 电子公告：[EB]
- 互联网络：[OL]

示例：

```text
[8] Herry R, Patel M. Application Profiles: Mixing and Matching Metadata Schemas[EB/OL]. http://www.ariadne.ac.uk/issue25/app-profiles/intro.html, 2012-09-10.
```

## 十二、正文写作要求

写作文献综述时应满足以下要求：

1. 主题明确：全文围绕主题展开，不离题。
2. 论证充分：有观点、有思路、有材料、有说服力。
3. 结论清楚：结论明确，便于理解。
4. 逻辑严密：前后联系紧密，避免杂乱无章。
5. 词简意明：文字简练，表达清楚。
6. 结构完整：绪言、主体、总结、参考文献四部分齐全。
7. 重点突出：不能平均用力，应突出关键问题和代表性文献。
8. 引用规范：正文引用与文后参考文献一一对应。
9. 图文结合：图、表、公式应服务于论述。
10. 避免抄袭：引用他人观点、数据、图表和结论时必须注明来源。

## 十三、推荐写作提纲模板

以下提纲可直接用于本次文献综述大作业：

## 十四、适合放入文章的图表类型

为了满足“图文并茂”的要求，建议加入以下类型的图表：

### 1. 研究发展时间线图

用于展示该研究方向从早期方法到最新方法的发展脉络。

### 2. 技术路线分类图

用于展示不同研究方法之间的分类关系，例如传统方法、优化方法、学习方法、混合方法等。

### 3. 文献统计图

可统计近十年文献数量变化、不同研究方向文献占比、国内外研究数量对比等。

### 4. 代表性方法对比表

表格可包含：作者、年份、研究对象、方法、数据来源、主要贡献、局限性等。

### 5. 问题与趋势总结表

可列出当前问题、产生原因、已有解决方案、未来改进方向。

## 十五、避免学术不端

写作中必须避免抄袭和不规范引用。抄袭通常指将他人作品、观点、数据、图表或文字片段据为己有而不注明来源。

需要特别注意：

1. 引用他人正式发表成果时必须标注文献来源。
2. 使用他人图表、数据或方法时必须注明出处。
3. 不能复制他人文字后简单替换个别词语。
4. 不能使用未阅读过的“第二手文献”充当自己参考过的文献。
5. 不能编造数据、实验结果或参考文献。
6. 不能大段照搬已有论文的结构和表述。
7. 参考文献必须与正文引用对应。

## 十六、最终检查清单

提交前可按以下清单检查：

```text
□ 题目是否具体、准确、与正文一致？
□ 是否包含绪言、主体、总结、参考文献四个基本部分？
□ 字数是否在 4000—8000 字之间？
□ 是否有图、表或必要公式？
□ 图题是否在图下方？
□ 表题是否在表上方？
□ 图表是否在正文中被引用和解释？
□ 参考文献是否不少于 30 篇？
□ 参考文献是否一般不超过 50 篇？
□ 近 5 年文献是否不少于一半？
□ 正文引用编号是否连续？
□ 文后参考文献格式是否规范？
□ 是否避免使用百度知道、论坛等非权威来源？
□ 是否避免引用未阅读过的第二手文献？
□ 是否在结论中总结成果、问题和未来趋势？
□ 是否避免文献简单罗列？
□ 是否体现了自己的分析和评价？
□ 文件命名是否为“学号-姓名-科技写作大作业”？
```

## 十七、一句话写作原则

本次大作业不是简单介绍“有哪些论文”，而是要围绕一个明确主题，把已有研究“分类整理清楚、发展脉络讲清楚、代表成果评清楚、存在问题说清楚、未来趋势写清楚”。
